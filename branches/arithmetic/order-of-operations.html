Order of Operations.

Rules which express what operation to perform first.

Definitions.

Operator: A mathematical symbol which is used to represent some behavior/effect. For example '+' & '-' are operators since they represent addition and subtraction respectively.

Operand: Any number or variable.

Variable: A letter which is used to represent some unknown value.

E.g
a) Given the equation, \(x + 7 = 4\), there are 3 operands, and 1 operators.
Operands: 'x', '7', '4'
Operators: '+'

b) Given the equation, \(y - 2 + 3 - 8 = 0\), there are 5 operands, and 3 operators.
Operands: 'y', '2', '3', '8', '0'
Operators: '-', '+', '-'

Key Concept.

The order of operations are a set of rules that inform which mathematical operation to perform first. Luckily, there is an easy way to remember which operations to perform first — BEDMAS.
