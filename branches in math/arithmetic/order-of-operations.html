Order of Operations.

Rules which express what operation to perform first.

Definitions.

Operator: A mathematical symbol which is used to represent some behavior/effect. For example '+' & '-' are operators since they represent addition and subtraction respectively.

Operand: A number or variable which is used to perform some type of operation.

Variable: A letter which is used to represent some value based on the contect of a Mathematical problem. It is usually used to perform some type of operation.

Number: A value/amount that is usually used to perform some type of operation.

E.g
a) Given the equation, \(ax^2 + bx + c = 0\), there are 4 operands, and 2 operators.
Operands: 'a', 'b', 'c', '0'
Operators: '+', '+'

b) Given the equation, \(x + 7 = 4\), there are 3 operands, and 1 operators.
Operands: 'x', '7', '4'
Operators: '+'

c) Given the equation, \(y - 2 + 3 - 8 = 0\), there are 5 operands, and 3 operators.
Operands: 'y', '2', '3', '8', '0'
Operators: '-', '+', '-'
